# elements-policy

Privacy policy for **Elements** — the periodic-table study app (iOS, iPadOS, macOS, Apple Watch, and Android).

Published via GitHub Pages: <https://rizkcorsight.github.io/elements-policy/>

- `index.html` — English, authoritative.
- `<locale>/index.html` — convenience translations for every language the app ships in (52 locales). The English version controls if there is any difference.
- `build.py` — regenerates all pages from `translations.json` (run `python3 build.py`).

Elements collects nothing: no accounts, no analytics, no advertising, no network calls of its own, no cloud. Everything lives on the device.
