Patakaran sa privacy

Ang Elements ay isang app para sa pag-aaral ng periodic table na nasa-device lamang. Hindi ito nangangailangan ng account, hindi nagbebenta ng personal na data, hindi nagpapakita ng mga ad, at hindi gumagamit ng anumang analytics o pagsubaybay. Ang data na nilikha mo ay nananatili sa iyong device, sa pribadong storage ng app: ang iyong mga study deck at flashcard, ang iyong kasaysayan ng pagrerepaso at iskedyul ng spaced-repetition, ang iyong mga elementong na-star, at ang iyong mga setting.

Ano ang hindi namin kinokolekta

Hindi namin kinokolekta ang iyong pangalan, email, numero ng telepono, lokasyon, mga advertising identifier, mga contact, mikropono, o data ng paggamit. Walang mga SDK para sa advertising, analytics, attribution, o pagsubaybay.

Mga pagbili

Libre ang pag-download ng Elements na may 3 araw na buong access. Pagkatapos noon, isang opsyonal na isahang pagbili lamang ang permanenteng nagbubukas sa buong app — walang subscription, at walang awtomatikong nagre-renew o awtomatikong nagsisingil. Ang App Store (iOS) o Google Play (Android) ang humahawak sa pagbili; hindi nakikita ng Elements ang iyong pangalan, payment card, o mga detalye ng pagsingil. Ang pagsisimula ng 3-araw na access na ito ay nangyayari nang lokal sa iyong device at hindi gumagawa ng subscription o singil; nakikipag-ugnayan lang sa store kung pipiliin mong bilhin o i-restore ang unlock.

Paggamit ng network

Gumagana ang Elements nang hindi nagpapadala ng iyong data sa anumang server — wala itong server. Ang buong periodic table ay nakapaloob sa app, at ang normal na paggamit ay nananatili sa iyong device.

Mga feature na nasa-device

Ang binibigkas na mga pangalan ng elemento ay gumagamit ng built-in na text-to-speech ng iyong device (audio output lamang — walang mikropono, walang naitatala). Ang opsyonal na explainer, kapag available, ay tumatakbo sa iyong device gamit ang on-device na modelo ng Apple at walang ina-upload. Ang Apple Watch sync ay nagpapalitan ng iyong progreso nang direkta sa pagitan ng iyong iPhone at Watch, hindi kailanman sa pamamagitan ng server. Gumagamit ang Wear OS app ng naka-bundle na data ng mga elemento sa relo at hindi nagsi-sync sa anumang developer server o cloud.

Mga pahintulot

Anumang pahintulot ay ginagamit lamang para sa feature na pipiliin mo, gaya ng opsyonal na paalala sa pag-aaral o ang widget sa home screen. Maaari mong baguhin ang mga ito sa mga setting ng iyong device.

Mga bata

Ang Elements ay nakarating para sa lahat, walang nilalamang nakakasakit, at hindi nangongolekta ng anumang personal na impormasyon mula sa kahit sino — kabilang ang mga bata.

Mga pagbabago

Kung magbabago ang patakarang ito, ipo-post namin dito ang na-update na bersyon at babaguhin ang petsa sa itaas.
