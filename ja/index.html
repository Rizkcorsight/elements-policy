プライバシーポリシー

Elements は、すべての処理を端末上で行う周期表の学習アプリです。アカウントは不要で、個人データを販売することはなく、広告を表示せず、分析やトラッキングも一切行いません。あなたが作成したデータは、アプリの非公開ストレージ内の端末上にとどまります。学習デッキやフラッシュカード、復習履歴と間隔反復のスケジュール、お気に入りの元素、各種設定がこれに含まれます。

収集しない情報

私たちは、お名前、メールアドレス、電話番号、位置情報、広告識別子、連絡先、マイク、利用データを収集しません。広告、分析、アトリビューション、トラッキングのための SDK は一切ありません。

購入について

Elements は無料でダウンロードでき、3日間すべての機能をご利用いただけます。その後は、任意で一度きりの購入を行うことでアプリ全体が永続的にアンロックされます。サブスクリプションはなく、自動更新や自動課金も一切ありません。購入は App Store（iOS）または Google Play（Android）が処理し、Elements がお名前、お支払いカード、請求情報を確認することはありません。

ネットワークの利用

Elements は、あなたのデータをいかなるサーバーにも送信せずに動作します。そもそもサーバーが存在しません。周期表のすべてがアプリ内に同梱されており、通常の利用はあなたの端末上で完結します。

端末上の機能

元素名の読み上げには、端末に内蔵されたテキスト読み上げ機能を使用します（音声出力のみで、マイクは使用せず、録音も行いません）。任意の解説機能は、利用可能な場合、Apple の端末上モデルを使ってあなたの端末上で動作し、何もアップロードしません。Apple Watch との同期は、あなたの進捗を iPhone と Watch の間で直接やり取りするもので、サーバーを経由することはありません。

権限について

いかなる権限も、任意の学習リマインダーやホーム画面ウィジェットなど、あなたが選んだ機能のためにのみ使用されます。これらは端末の設定でいつでも変更できます。

お子様について

Elements は全年齢対象として評価されており、不適切なコンテンツを含まず、お子様を含むいかなる人からも個人情報を収集しません。

変更について

本ポリシーを変更する場合は、更新版をこちらに掲載し、上記の日付を改訂します。
