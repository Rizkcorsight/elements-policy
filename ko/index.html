개인정보 처리방침

Elements는 기기에서만 작동하는 주기율표 학습 앱입니다. 계정이 필요 없고, 개인 데이터를 판매하지 않으며, 광고를 표시하지 않고, 분석이나 추적을 사용하지 않습니다. 사용자가 만든 데이터는 기기 내 앱의 비공개 저장 공간에 그대로 보관됩니다. 학습 덱과 플래시카드, 복습 기록과 간격 반복 일정, 즐겨찾기한 원소, 그리고 설정이 여기에 포함됩니다.

수집하지 않는 정보

당사는 사용자의 이름, 이메일, 전화번호, 위치, 광고 식별자, 연락처, 마이크, 사용 데이터를 수집하지 않습니다. 광고, 분석, 어트리뷰션, 추적용 SDK가 일절 없습니다.

구매

Elements는 무료로 다운로드할 수 있으며 3일간 모든 기능을 전체 이용할 수 있습니다. 그 이후에는 선택 사항인 일회성 구매 한 번으로 앱 전체를 영구적으로 잠금 해제합니다. 구독이 없으며, 자동 갱신이나 자동 청구도 발생하지 않습니다. 구매는 App Store(iOS) 또는 Google Play(Android)에서 처리하며, Elements는 사용자의 이름, 결제 카드, 청구 정보를 전혀 확인하지 않습니다.

네트워크 사용

Elements는 사용자의 데이터를 어떤 서버로도 전송하지 않고 작동합니다. 서버 자체가 없습니다. 주기율표 전체가 앱 안에 포함되어 제공되며, 일반적인 사용은 모두 기기 내에서 이루어집니다.

기기 내 기능

원소 이름 음성 읽기는 기기에 내장된 텍스트 음성 변환 기능을 사용합니다(오디오 출력만 사용하며, 마이크를 쓰지 않고 아무것도 녹음하지 않습니다). 선택 사항인 설명 기능은 제공되는 경우 Apple의 기기 내 모델을 사용해 기기에서 실행되며 아무것도 업로드하지 않습니다. Apple Watch 동기화는 사용자의 진행 상황을 iPhone과 Watch 사이에서 직접 주고받으며, 서버를 거치지 않습니다.

권한

모든 권한은 사용자가 선택한 기능에만 사용됩니다. 예를 들어 선택 사항인 학습 알림이나 홈 화면 위젯이 이에 해당합니다. 이러한 권한은 기기 설정에서 변경할 수 있습니다.

아동

Elements는 모든 연령에 적합한 등급을 받았으며, 부적절한 콘텐츠를 포함하지 않고, 아동을 포함한 그 누구로부터도 개인정보를 수집하지 않습니다.

변경 사항

본 방침이 변경되면 업데이트된 버전을 이곳에 게시하고 위의 날짜를 수정합니다.
